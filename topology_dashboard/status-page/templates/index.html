Como funciona

O proxy reverso mantém uma URL única de acesso. Os dois frontends entregam a interface web em redundância. Um banco PostgreSQL compartilhado coordena o cluster HA dos dois Zabbix Server — só o nó ACTIVE executa as coletas, enquanto o outro fica em STANDBY. Se o ativo cair, o standby assume a coleta automaticamente; o host de teste no fim da cadeia prova essa continuidade em tempo real.
